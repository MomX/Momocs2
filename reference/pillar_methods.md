# Pillar support for morphometric data types

These functions provide custom display methods for morphometric classes
in tibble printing via the pillar package.

## Usage

``` r
type_sum.coo(x)

type_sum.out(x)

type_sum.ldk(x)

type_sum.ldk_id(x)

type_sum.path(x)

type_sum.cur(x)

type_sum.dct(x)

type_sum.eft(x)

type_sum.npoly(x)

type_sum.opoly(x)

type_sum.proc(x)

# S3 method for class 'coo'
pillar_shaft(x, ...)

# S3 method for class 'out'
pillar_shaft(x, ...)

# S3 method for class 'ldk'
pillar_shaft(x, ...)

# S3 method for class 'ldk_id'
pillar_shaft(x, ...)

# S3 method for class 'path'
pillar_shaft(x, ...)

# S3 method for class 'cur'
pillar_shaft(x, ...)

# S3 method for class 'dct'
pillar_shaft(x, ...)

# S3 method for class 'eft'
pillar_shaft(x, ...)

# S3 method for class 'npoly'
pillar_shaft(x, ...)

# S3 method for class 'opoly'
pillar_shaft(x, ...)

# S3 method for class 'proc'
pillar_shaft(x, ...)
```
