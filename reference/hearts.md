# Heart dataset

A list of 4 shapes (as outlines)

## Usage

``` r
hearts
```

## Format

A data frame with 40 rows and three columns

- img:

  dummy paths

- coo:

  outline coordinates

- coo_ldk:

  landmark id (4) on these coordinates

- author:

  who drawn these

## Source

We thank the fellows of the Ecology Department of the French Institute
of Pondicherry circa 2011 that drawn the hearts. They then have been
landmarked, smoothed, scaled, centered, and downsampled to 80
coordinates per outline.
