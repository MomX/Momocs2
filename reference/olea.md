# Olea dataset

An open curves dataset with two views derived from the legacy Momocs
olea dataset.

## Usage

``` r
olea
```

## Format

A data frame with 90 rows (three accessions):

- id:

  unique id

- VD:

  curves coordinates for dorsal view

- VL:

  curves coordinates for lateral view

- var:

  olive accession name

- status:

  factor, whether accession is wild or cultivated

## Source

We thank Jean-Frederic Terral and Sarah Ivorra (UMR CBAE, Montpellier,
France) from allowing us to share the data.

You can have a look to the original paper: Terral J-F, Alonso N,
Capdevila RB i, Chatti N, Fabre L, Fiorentino G, Marinval P, Jorda GP,
Pradat B, Rovira N, et al. 2004. Historical biogeography of olive
domestication (*Olea europaea* L.) as revealed by geometrical
morphometry applied to biological and archaeological material. *Journal
of Biogeography* 31: 63-77.

## Details

Olive stones, photographed using two orthogonal views.
